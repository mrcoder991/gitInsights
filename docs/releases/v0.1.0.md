# v0.1.0 — first public cut

Shipped because the dashboard finally feels honest: your commits, your story, not a manager heatmap cosplay.

## What you get

- GitHub OAuth in the browser, analytics on your machine, optional gist sync if you want the same headspace on two laptops.
- Dashboard bento: heatmap, streaks, WLB audit copy that tells you when to log off, holidays off-days from bundled public calendars, settings that round-trip through export/import.
- GitHub Pages app plus Vercel token exchange — see the root README for env wiring.

## Notes

- Public `/u/:username` is still a stub until the spec §11 call lands.
- After deploy, paste your Pages URL through the usual card validators if Slack or X is picky about cache.

Tag when ready: `git tag -a v0.1.0 -m "v0.1.0 first public cut" && git push origin v0.1.0`.
